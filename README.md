# Amy Art - Landing Page

## Project Title
Amy Art - Charcoal & Graphite Sketches Landing Page

## Objective
The objective of this project is to design and develop a modern, responsive, and visually appealing landing page for an art portfolio. The website showcases artwork, artist information, and contact details with a clean user interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome

## Project Files

- Adityakumar_1.html
- Adityakumar_1.css
- Adityakumar_1.js
- assets/
- images/

## Features
- Responsive Landing Page
- Hero Section
- Artwork Gallery
- About Section
- Contact Section
- Smooth Scrolling Navigation
- Professional UI Design
- Mobile Friendly Layout

## Steps Performed
1. Created the website structure using HTML.
2. Designed the user interface using CSS.
3. Added interactivity using JavaScript.
4. Tested responsiveness on different screen sizes.
5. Optimized layout and images for better user experience.

## How to Run
1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open **Adityakumar_1.html** in your browser or run it using the Live Server extension.

## Outcome
Successfully developed a responsive landing page with a modern design and smooth user experience.
